Система мониторинга твердотопливного котла

Описание: Система предназначена для мониторинга температуры теплоносителя твердотопливного котла (дополнительно подключены датчики температуры улицы, котельной и подвала дома), оповещения пользователя при достижении температуры критического значения. Также имеется возможность удаленного управления насосом, ТЭНом котла. Для визуального контроля подключена веб-камера.

Проделанная работа: Настройка ОС и веб-сервера, подключение температурных датчиков и реле управления, разработка и верстка дизайна, разработка бэкенда, вывод графика изменения температуры за указаный период, настройка OpenVPN, создание Android-приложения

Используемые технологии: PHP, MySql, JQuery, ChartJS, Bootstrap, bash, OpenVPN, Java
